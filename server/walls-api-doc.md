# Wall API Stuff

### Get Wall Metadata and Contents
`/api/v1/get-wall/{wall-id}`

### Create Wall
`/api/v1/create-wall/{user}/{width}/{height}`

### Get Pixel
`/api/v1/get-pixel/{wall-id}/{x}/{y}`

### Set Pixel
`/api/v1/set-pixel/{wall-id}/{x}/{y}/{hex-colour}/{user}`

### Get Wall Preview
`/api/v1/get-preview/{wall-id}`
